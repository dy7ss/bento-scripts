# JSON フォーマッター

改行されていないJSONを入力して、フォーマッティングされたJSONに変換するWebツールです。

## 機能

- **JSON整形**: 改行されていないJSONを見やすく整形します
- **エラー検出**: 無効なJSONの場合、エラーメッセージを表示します
- **コピー機能**: フォーマッティング済みのJSONをクリップボードにコピーできます
- **静的Webサイト**: HTMLのみで動作し、外部サーバーは不要です
- **レスポンシブデザイン**: パソコン、タブレット、スマートフォンで利用できます

## 使用方法

1. `index.html`をWebブラウザで開く
2. 左側のテキストエリアに改行されていないJSONを入力
3. 「フォーマット」ボタンをクリック（または Ctrl+Enter）
4. 右側に整形されたJSONが表示されます
5. 「コピー」ボタンで結果をコピーできます

## ファイル構成

- `index.html` - メインのHTMLファイル
- `styles.css` - スタイルシート
- `script.js` - JavaScriptロジック
- `README.md` - このファイル
