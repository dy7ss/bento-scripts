# bulk-startup

このスクリプトは、Visual Studio Code（VSCode）で複数のファイルやディレクトリを一括で開くための機能を提供します。

## 構成ファイル

- `code2.sh`  
  引数で指定した複数のファイルやディレクトリをVSCodeで順に開くシェル関数です。

## 使い方

1. `code2.sh` の内容を `.bashrc` や `.zshrc` などの設定ファイルにコピー＆ペーストしてください。

    ```sh
    # 例: .bashrc の末尾に追記
    code2() { for N in "$@"; do code "$N"; done; }
    ```

2. ターミナルを再起動するか、設定ファイルを再読み込みしてください。

    ```sh
    source ~/.bashrc
    ```

3. `code2` コマンドとして利用できます。  
   例: 複数のファイルやディレクトリを一度に開く

    ```sh
    code2 dir1 dir2 dir3
    ```

   上記コマンドで、`dir1`、`dir2`、`dir3` がVSCodeで順に開かれます。

## 注意事項

- VSCodeのコマンドラインツール `code` がインストールされている必要があります。
- 引数に指定した順にVSCodeで開きます。
- シンプルな関数のため、エラーハンドリング等はありません。
